# shiftwaltz 쉬프트왈츠

**쉬프트왈츠**는 홈로우를 쓰기 힘든 한국 두벌식 타이핑 환경 및 그래픽 디자이너를 위한 36키 키맵입니다.

- 한영 키에 해당되는 오른쪽 Alt가 기본 레이어에 들어가있습니다.
- 왼쪽 Ctrl키도 기본 레이어에 들어가 있습니다. 단축키의 편의성을 위함입니다.
- 숫자열과 F키 열을 텐키리스처럼 사용합니다. 키패드도 좋지만 양손 조작은 이쪽이 편하더군요.
- 기본적으로 HOLD_ON_OTHER_KEY_PRESS같은 기능을 켜두는 걸 추천드립니다. 좀 더 빠른 타이핑 환경을 구성할 수 있습니다.

## 기본 레이어

![L0](file://C:\projects\git\shiftwaltz\images\shiftwaltz-layer0.png?msec=1762612601115)

아래에 붙어있는 음각은 길게 누르면 그 키로 바뀌는 것들입니다. 홀드 액션이라고도 합니다. 썸 클러스터라고도 하는 엄지 키 부분에 대부분 모여있습니다.

두벌식 자판을 사용하니 일반적인 쿼티 구성으로 영문을 구성해두었습니다. 홈로우 대신에 비슷한 포지션으로 ,./키에 다른 모디파이어를 홀드키로 두었습니다.

키보드나 사용자에 따라선 썸클러스터 중간에 Shift를 두는 게 편할 때가 있고, 가장 바깥에 두는 게 편할 때가 있습니다. 원하시는 대로 홀드 키만 바꿔서 위치시키시면 됩니다.

참고로, 왼손 b키 문제 말인데... 세미콜론; 대신에 b키 넣어도 꽤 나쁘지 않더라고요. 물음표 자리의 LGUI 빼면 그렇게까지 필요하진 않으니, 적당히 아랫열을 모디키와 안쪽 내용물을 수정하셔도 좋습니다. nm,./ ->bnm,.을 써도 좋고요.
